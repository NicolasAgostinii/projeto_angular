O — OBJETIVO

Desenvolver uma aplicação Web CRUDL utilizando o framework Angular.

O tema escolhido para a aplicação é:

Moda e inspiração de looks

A ideia da aplicação é criar uma plataforma inspirada no Pinterest, onde usuários podem compartilhar imagens relacionadas à moda.

O sistema deverá permitir:

Cadastro de publicações de moda;
Visualização das publicações;
Edição das informações;
Exclusão de publicações;
Listagem dos conteúdos cadastrados.

Cada publicação deverá possuir informações como:

Imagem;
Nome/título da publicação;
Descrição;
Categoria ou estilo (opcional).

Exemplo:

Nome:
"Look casual de inverno"

Imagem:
foto do look

Descrição:
"Combinação usando jaqueta jeans e botas para dias frios."
